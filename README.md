# Curatescape Galleries

Adds options for customizing the display of media files on Item and File record pages. In compatible themes, media files can be presented in multiple customizable, accessible media layouts and displayed in an interactive [PhotoSwipe](https://photoswipe.com/) lightbox, with various improvements to the display of audio, video, and PDF documents.

# FAQ

### Why don't the File Display Settings change anything in my theme?

Each theme is created differently. Some themes – such as Big Picture, Center Row, and The Daily – already have their own custom file viewers with their own display options.

### Why are PDF files not fully interactive? How can I change the way PDF files are displayed?

PDFs can be displayed inline in Gecko- and Chromium-based browsers including Chrome, Edge, Opera, and Firefox. In those case, PDF files will be displayed using the native browser-based PDF viewer. Safari support is limited at this time. You may disable or modify PDF-related functionality in plugin settings.

### Why are some files not showing up?

Some gallery types may not be compatible with certain types of files. For the best results, limit files to common web-friendly formats such as JPG, PNG, MP3, MP4, and PDF. Choose the gallery type that works best across all the different types of items on your site.

### Can I choose a gallery style for a specific item?

Not at this time, though that is under consideration for a future version.
